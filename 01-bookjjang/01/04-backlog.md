## 진행 순서 (권장 스프린트)
```
1주차: Epic 0 (S1~S4)  +  Epic 1 (A1~A3)          → 로그인해서 빈 홈 진입
2주차: Epic 2 (B1~B5)                             → 검색해서 서재에 책 추가
3주차: Epic 3 (C1~C4)  +  Epic 4 (D1~D2)          → 기록 작성 + 목록 확인
4주차: Epic 4 (D3~D4)  +  QA/버그픽스              → 통계까지 완성 = MVP
```

## Phase 1 완료(Definition of Done)
- [ ] 구글 로그인 후 빈 서재 온보딩이 보인다.
- [ ] 알라딘으로 책을 검색해 상세를 보고 서재에 추가할 수 있다(중복 방지).
- [ ] 책마다 별점·한줄평·메모를 남기고 다시 열어 확인할 수 있다.
- [ ] 내 서재에서 읽은 책 목록과 통계(권수·평균 별점·월별 그래프)를 볼 수 있다.
- [ ] 모든 사용자 데이터는 RLS로 본인만 접근 가능하다.
- [ ] 반응형으로 모바일/데스크톱에서 정상 동작한다.

> Phase 2(시작일/완료일·진행률·상태 분류·프로필 편집·정렬/필터)와 Phase 3(목표·장르 통계·내보내기·스트릭)은 별도 백로그로 이어감.
