## 1. 정보구조 (IA)

```
북박이장
├─ 로그인 (비로그인 진입점)
│    └─ 구글 로그인
│
├─ 마이페이지 = 내 서재 (로그인 후 홈)
│    ├─ 프로필 (별명 · 프로필 이미지)
│    ├─ 독서 통계 (총 권수 · 읽는 중 · 평균 별점 · 월별 그래프)
│    └─ 읽은 책 목록 (그리드/리스트 · 정렬·필터)
│
├─ 책 검색
│    ├─ 검색창 (제목/저자)
│    ├─ 검색 결과 리스트
│    └─ 책 상세 → [내 서재에 추가]
│
└─ 독서 기록 (서재의 책 1권 상세)
     ├─ 별점 (1~5)
     ├─ 한줄평
     ├─ 메모 (책당 1개)
     ├─ 독서 상태 · 시작일/완료일
     └─ 진행률
```

**전역 네비게이션(로그인 후)**: `내 서재` · `책 검색` · `프로필(별명/이미지)`

## 2. 화면 흐름 (User Flow)

```
        ┌────────────┐
        │  로그인     │  ── 구글 로그인 ──┐
        └────────────┘                   ▼
                              ┌─────────────────────┐
              ┌────────────── │  마이페이지 / 내 서재 │ ◀──────────┐
              │               └─────────────────────┘             │
              │                    │            │                  │
       [책 검색] 이동          [책 카드 클릭]  [프로필 편집]         │
              ▼                    ▼            ▼                  │
        ┌───────────┐      ┌────────────┐  ┌───────────┐          │
        │ 책 검색    │      │ 독서 기록   │  │ 프로필 수정 │─ 저장 ─┘
        └───────────┘      │ (책 상세)   │  └───────────┘
              │            └────────────┘
        [검색→결과→상세]         │
              ▼             (별점/한줄평/메모/
        ┌───────────┐        상태/진행률 저장)
        │ 책 상세    │             │
        │[서재에 추가]│─── 추가 ───▶ 내 서재로 반영
        └───────────┘
```

**핵심 경로 (Happy Path)**: 로그인 → 책 검색 → 상세 → 서재에 추가 → 독서 기록 작성 → 내 서재에서 통계 확인. (목표: 3분 이내 첫 기록)

## 5. 상태·예외 처리 원칙
- **비로그인** 보호 페이지 접근 → 로그인으로 리다이렉트.
- **검색 0건 / API 오류** → 빈 상태·재시도 안내.
- **중복 추가** → "이미 서재에 있어요" 안내 후 해당 기록으로 이동.
- **빈 서재** → 온보딩 카피 + [책 검색하러 가기] CTA.
